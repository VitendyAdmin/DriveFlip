# Design Mockups

Static HTML mockups used during development to prototype UI layouts before implementing them in WPF/XAML.

## Files

- **mockups.html** - 3-pane layout mockup created during the UI restructure from the original 2-column design to the current 3-column layout (drive list | details | actions). Used as a visual reference for spacing, colors, and component placement before writing XAML.

## Notes

- These are design-time artifacts, not shipped with the application.
- Colors and layout in the mockups may be slightly outdated compared to the final XAML implementation.
- Open any `.html` file directly in a browser to view.
